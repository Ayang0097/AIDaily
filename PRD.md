# AI日报 APP - 产品需求文档 (PRD)

## 1. 产品概述

### 产品名称
**AI日报** (AI Daily Digest)

### 产品定位
聚焦AI领域的资讯聚合与解读平台，为AI从业者、开发者、投资人提供一站式的每日资讯消费体验。

### 核心价值
- **省时间**：一篇看完今日所有AI重要动态
- **有深度**：不是标题党，有观点有分析
- **可追溯**：历史存档，方便回顾

### 目标用户
- AI/ML 工程师、研究员
- AI 创业者、投资人
- 对AI感兴趣的大众用户

---

## 2. 功能需求

### 2.1 资讯聚合
- **来源**：虎嗅、爱范儿、钛媒体、少数派、Solidot、36kr、Twitter/X AI KOL
- **抓取**：RSS订阅 + Twitter监控
- **去重**：基于标题+链接去重
- **更新频率**：每日早7点自动生成日报

### 2.2 日报生成
- **内容组织**：按主题分类（模型发布、行业动态、技术突破、工具推荐、观点评析）
- **AI摘要**：每条资讯生成100字摘要
- **推荐指数**：⭐1-5星标记重要程度
- **一句话点评**：AI生成犀利短评
- **AI问答**：基于资讯内容的智能问答

### 2.3 用户功能
| 功能 | 描述 | 状态 |
|------|------|------|
| 今日日报 | 查看当日完整日报 | ✅ |
| 历史日报 | 按日期浏览历史日报 | ✅ |
| 收藏 | 收藏单条资讯 | ✅ |
| 分享 | 分享日报到微信/链接 | ✅ |
| 推送 | 微信/WhatsApp定时推送 | ✅ |
| 搜索 | 搜索历史资讯 | ✅ |
| 深色模式 | 护眼深色主题 | ✅ |
| 下拉刷新 | 下拉刷新最新内容 | ✅ |
| AI问答 | 基于资讯的智能问答 | ✅ |
| 快速统计 | 今日阅读/收藏统计 | ✅ |

### 2.4 管理员功能
| 功能 | 描述 |
|------|------|
| 日报管理 | 编辑/删除/重新生成日报 |
| 来源配置 | 添加/删除RSS源 |
| 推送管理 | 配置推送时间/渠道 |
| 数据统计 | UV/PV/用户数统计 |

## 3. 技术方案

### 3.1 前端
- **Web App**：React + TailwindCSS（响应式）
- **移动端**：React Native（后续）
- **状态管理**：Zustand
- **构建工具**：Vite

### 3.2 后端
- **Runtime**：Node.js
- **框架**：Express
- **数据库**：SQLite（轻量）/ PostgreSQL（生产）
- **ORM**：Prisma

### 3.3 数据管道
```
RSS/Twitter → 抓取 → 解析 → 去重 → 存储
                              ↓
                      AI摘要生成(GPT-4)
                              ↓
                      日报组装 → 推送
```

### 3.4 推送渠道
- 微信公众号（草稿推送）
- WhatsApp（个人助理）
- 邮件（可选）

## 4. API 设计

### 公开接口
| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/daily/{date} | 获取指定日期日报 |
| GET | /api/daily/latest | 获取最新日报 |
| GET | /api/sources | 获取资讯来源列表 |
| GET | /api/search?q={keyword} | 搜索资讯 |

### 管理接口（需认证）
| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/admin/regenerate | 重新生成指定日期日报 |
| DELETE | /api/admin/daily/{date} | 删除日报 |
| PUT | /api/admin/sources | 更新来源配置 |
